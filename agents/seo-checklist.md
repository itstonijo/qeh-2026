# On-Page SEO Checklist

The `marketing-seo` agent audits the site against this list.

## Per page
- [ ] **Title tag** — unique, ≤ 60 chars, primary keyword near the front.
- [ ] **Meta description** — compelling, 120–158 chars, includes the keyword and a reason to click.
- [ ] **One H1** — matches search intent; logical H2/H3 hierarchy below it.
- [ ] **Canonical URL** set.
- [ ] **Open Graph** tags (title, description, url, image) for social sharing.
- [ ] **Twitter card** tags.
- [ ] **JSON-LD structured data** that parses and matches the page's real content.
- [ ] **Image alt text** on every meaningful image.
- [ ] **Descriptive, readable internal links** (no "click here").
- [ ] **Mobile-friendly** viewport + responsive layout.

## Site-wide
- [ ] `robots.txt` present and not blocking indexable pages.
- [ ] `sitemap.xml` present, listing every real page, referenced from `robots.txt`.
- [ ] No broken internal links.
- [ ] Fast load (minimise/critical CSS, compressed images).
- [ ] HTTPS canonical URLs.

## Content & campaigns
- [ ] One clear primary keyword + a few secondary keywords per page, used naturally.
- [ ] Copy is truthful and specific — no unverifiable claims.
- [ ] Outbound ad/campaign links carry correct UTM parameters.
- [ ] Every campaign logged in `campaigns/campaigns.json`.
